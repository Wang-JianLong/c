1. 普通变量存放的是数据 指针变量存放的是地址内存地址
2. 声明
	- 数据类型 * 变量名
	- char *p 定义一个指向字符型的指针变量 
3. 使用

```c
int a = 10;
// & 取址运算符
int *ap = &a;
// 访问指针所引向的数据
int c = *ap;
```
4. 禁止访问为初始化的指针(野指针)
```c
int *p;

*p = 123;
```
5. 注意区分左值与常量
6. 数组名只是一个地址，而指针是左值
7. 指针数组是数组 数组指针是指针
	- int *p1[5]  指针数组 数组元素是整形指针
	- int (*p1)[5]
